## 🚀 Основные проекты

### 🏦 [LakePay](https://github.com/maydeixxx/lakepay)
**Микросервисная платформа для торговли игровыми аккаунтами**

🔹 **Tech Stack:** Java 21 | Spring Boot 3.4 | Kafka | Redis | PostgreSQL | Docker  
🔹 **Архитектура:** 5 микросервисов (Gateway, Eureka, Auth, Payment, Chat)  
🔹 **Особенности:**
- Интеграция Telegram Wallet для криптоплатежей
- Real-time чат через WebSocket (STOMP)
- JWT authentication + роли
- Event-driven через Apache Kafka
- Трёхслойная архитектура (Entity-Domain-DTO)
- BigDecimal для финансовых операций

### 💳 [OmniPoint](https://github.com/maydeixxx/loyaltyEngine)
**Event-driven система лояльности и кэшбэка**

🔹 **Tech Stack:** Java 21 | Spring Boot 3.x | Kafka | Redis | PostgreSQL  
🔹 **Архитектура:** SAGA Choreography pattern (3 микросервиса)  
🔹 **Особенности:**
- Dynamic Rule Engine для расчёта кэшбэка
- Kafka resilience (Retry mechanism + Dead Letter Queue)
- Optimistic Locking для предотвращения Race Condition
- Redis кэширование с TTL для оптимизации производительности
- RestControllerAdvice для централизованной обработки ошибок

### 🕷️ Web Scraping Solutions
**High-performance веб-скраперы**

🔹 **Tech Stack:** Java | OkHttp3 | Jsoup | ExecutorService  
🔹 **Особенности:**
- Многопоточная обработка (100+ запросов/сек)
- Thread-safe коллекции (ConcurrentHashMap, LinkedBlockingQueue)
- Обработка Race Condition через атомарные операции
- Connection pooling в OkHttp3
